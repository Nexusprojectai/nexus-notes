# 📝 Nexus Notes

A fully capable, installable **notes app PWA** with folders, search, pinning, auto-save, and a beautiful dark neon theme.

## 🚀 Play Online

👉 **[Open Nexus Notes](https://nexusprojectai.github.io/nexus-notes/)**

## 📲 Install on Android

1. Open the link above in **Chrome**
2. Tap the **⋮ menu** → **Add to Home screen**
3. It'll install like a native app with its own icon

## ✨ Features

- 📁 **Folders** — organize notes into custom categories
- 🔍 **Search** — instant full-text search across titles and content
- 📌 **Pin notes** — keep important notes at the top
- 💾 **Auto-save** — every keystroke saved to IndexedDB instantly
- 🌙 **Dark neon theme** — easy on the eyes, beautiful glow effects
- 📱 **Responsive** — 3-panel desktop layout, single-column mobile
- 🔌 **Offline first** — service worker caches everything
- 🏠 **Installable PWA** — native feel on Android (and desktop)

## 🧑‍💻 Tech

Vanilla JS · IndexedDB · CSS3 · PWA (Service Worker + Manifest)
